# Data Jobs Dashboard (Power BI)

Portfolio-ready Power BI project analyzing global **data job trends** (job counts, geography, salary levels).

![Dashboard Preview](screenshots/dashboard-screenshot.png)

---

## ✨ Highlights
- **KPIs:** Total Job Count, Avg Yearly Salary, Avg Hourly Salary
- **Top Roles:** Bar chart (Data Engineer / Analyst / Data Scientist)
- **Geography:** Bubble map by latitude/longitude
- **Reproducible:** Sample CSV + PBIP project format (diff-friendly)

## 🧱 Tech Stack
- **Power BI Desktop** (PBIX/PBIP)
- **Power Query (M)**
- **DAX / Tabular model**
